## 2. 实验前准备

### 2.1 整理 GTSRB 测试集

原始测试集通常不是分类子目录结构，必须先执行：

```bash
python scripts/prepare_gtsrb_testset.py --data-root datasets
```

服务器上对应命令：

```bash
python /root/SBBiShe/scripts/prepare_gtsrb_testset.py --data-root /root/SBBiShe/datasets
```

### 2.2 Refool 反射图目录

Refool 需要单独准备反射图目录：

- 本地：`datasets/refool_reflections`
- 服务器：`/root/SBBiShe/datasets/refool_reflections`

## 3. 本地 Smoke Test

建议先跑 `BadNets -> REFINE -> eval` 一条最短链路。

```bash
python scripts/train_gtsrb_badnets.py --data-root datasets --gpu-id 0 --epochs 1 --disable-schedule --batch-size 16 --num-workers 0 --test-interval 1 --save-interval 1
python scripts/train_refine_gtsrb.py --attack badnets --data-root datasets --gpu-id 0 --epochs 1 --disable-schedule --batch-size 16 --num-workers 0 --test-interval 1 --save-interval 1
python scripts/eval_refine_gtsrb.py --attack badnets --data-root datasets --gpu-id 0 --batch-size 16
```

说明：

- 攻击脚本用 poisoned dataset 训练攻击模型
- `train_refine_gtsrb.py` 现在用 clean `trainset/testset` 训练 REFINE
- `eval_refine_gtsrb.py` 才在 poisoned testset 上评估 `ASR_NoTarget`

## 4. 服务器正式实验顺序

推荐按下面顺序执行：

1. benign 基线
2. 4 种攻击模型
3. 针对每种攻击训练 REFINE
4. 评估每种攻击对应的 REFINE

### 4.1 benign 基线

```bash
python /root/SBBiShe/scripts/train_gtsrb_benign.py \
  --data-root /root/SBBiShe/datasets \
  --experiment-root /root/SBBiShe/experiments \
  --gpu-id 0 --epochs 100 --batch-size 128 --num-workers 8 \
  --test-interval 10 --save-interval 10
```

### 4.2 四种攻击训练

把 `<PR>` 替换为目标投毒率，例如 `0.1`、`0.05`、`0.01`。

```bash
python /root/SBBiShe/scripts/train_gtsrb_badnets.py --data-root /root/SBBiShe/datasets --experiment-root /root/SBBiShe/experiments --gpu-id 0 --epochs 100 --batch-size 128 --num-workers 8 --test-interval 10 --save-interval 10 --poisoned-rate <PR>
python /root/SBBiShe/scripts/train_gtsrb_blended.py --data-root /root/SBBiShe/datasets --experiment-root /root/SBBiShe/experiments --gpu-id 0 --epochs 100 --batch-size 128 --num-workers 8 --test-interval 10 --save-interval 10 --poisoned-rate <PR>
python /root/SBBiShe/scripts/train_gtsrb_wanet.py --data-root /root/SBBiShe/datasets --experiment-root /root/SBBiShe/experiments --gpu-id 0 --epochs 100 --batch-size 128 --num-workers 8 --test-interval 10 --save-interval 10 --poisoned-rate <PR>
python /root/SBBiShe/scripts/train_gtsrb_refool.py --data-root /root/SBBiShe/datasets --reflection-dir /root/SBBiShe/datasets/refool_reflections --experiment-root /root/SBBiShe/experiments --gpu-id 0 --epochs 100 --batch-size 128 --num-workers 8 --test-interval 10 --save-interval 10 --poisoned-rate <PR>
```

### 4.3 REFINE 训练

REFINE 当前推荐口径是：

- 加载攻击模型 checkpoint
- 使用 clean GTSRB `trainset/testset` 训练 UNet
- 训练完成后再做防御评估

```bash

python /root/SBBiShe/scripts/train_refine_gtsrb.py --attack wanet --data-root /root/SBBiShe/datasets --experiment-root /root/SBBiShe/experiments --gpu-id 0 --epochs 150 --batch-size 128 --num-workers 8 --test-interval 10 --save-interval 10 --poisoned-rate <PR>
python /root/SBBiShe/scripts/train_refine_gtsrb.py --attack refool --data-root /root/SBBiShe/datasets --reflection-dir /root/SBBiShe/datasets/refool_reflections --experiment-root /root/SBBiShe/experiments --gpu-id 0 --epochs 150 --batch-size 128 --num-workers 8 --test-interval 10 --save-interval 10 --poisoned-rate <PR>
```

## 5. 推荐显式绑定 checkpoint

做多投毒率实验时，推荐显式指定 checkpoint，避免自动推断串组。

### REFINE 训练

```bash
python /root/SBBiShe/scripts/train_refine_gtsrb.py \
  --attack badnets \
  --data-root /root/SBBiShe/datasets \
  --experiment-root /root/SBBiShe/experiments \
  --attack-checkpoint /root/SBBiShe/experiments/attacks/BadNets/<your_attack_dir>/ckpt_epoch_100.pth \
  --gpu-id 0 --epochs 150 --batch-size 128 --num-workers 8 \
  --test-interval 10 --save-interval 10 --poisoned-rate <PR>
```

### REFINE 评估

```bash
python /root/SBBiShe/scripts/eval_refine_gtsrb.py \
  --attack badnets \
  --data-root /root/SBBiShe/datasets \
  --experiment-root /root/SBBiShe/experiments \
  --attack-checkpoint /root/SBBiShe/experiments/attacks/BadNets/<your_attack_dir>/ckpt_epoch_100.pth \
  --refine-checkpoint /root/SBBiShe/experiments/refine/BadNets/train/<your_refine_dir>/ckpt_epoch_150.pth \
  --arr-path /root/SBBiShe/experiments/refine/BadNets/train/<your_refine_dir>/label_shuffle.pth \
  --gpu-id 0 --batch-size 128 --poisoned-rate <PR>
```

## 6. 对论文最重要的对照组

至少保留下面两组：

1. `Attack only`
2. `Attack + REFINE(clean-data training)`

如果时间允许，建议再加一个消融组：

3. `Attack + REFINE(poisoned-data training, old setting)`

这样可以更清楚说明为什么之前出现“ASR 不下降”。

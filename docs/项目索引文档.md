# 项目索引文档

## 一、项目结构

```text
SBBiShe/
|-- core/
|   |-- attacks/      # BadNets / Blended / WaNet / Refool
|   |-- defenses/     # REFINE
|   |-- models/       # ResNet / UNet / UNetLittle
|   `-- utils/
|-- scripts/          # 所有训练/评估入口
|-- datasets/
|   |-- GTSRB/
|   `-- refool_reflections/
|-- experiments/
`-- docs/
```

## 二、核心脚本

- `scripts/prepare_gtsrb_testset.py`：将 GTSRB `testset` 从扁平结构整理为类别子目录。
- `scripts/train_gtsrb_benign.py`：训练 benign 基线。
- `scripts/train_gtsrb_badnets.py`：训练 BadNets 攻击。
- `scripts/train_gtsrb_blended.py`：训练 Blended 攻击。
- `scripts/train_gtsrb_wanet.py`：训练 WaNet 攻击。
- `scripts/train_gtsrb_refool.py`：训练 Refool 攻击。
- `scripts/train_refine_gtsrb.py`：训练 REFINE。
- `scripts/eval_refine_gtsrb.py`：评估 REFINE。

## 三、数据与格式约束

1. `train/` 必须是按类别分目录。
2. `testset/` 原始下载通常是扁平目录，必须先执行准备脚本。
3. 支持图片扩展名：`.ppm/.png/.jpg/.jpeg/.bmp`。

## 四、攻击/防御能力

- 攻击：BadNets / Blended / WaNet / Refool
- 防御：REFINE
- 模型：ResNet18（分类器）+ UNetLittle（REFINE 预处理模块）

## 五、路径规范（服务器）

- 项目根目录：`/root/SBBiShe`
- 数据目录：`/root/SBBiShe/datasets`
- 实验目录：`/root/SBBiShe/experiments`

## 六、参数规范

- `--attack` 统一用小写：`badnets/blended/wanet/refool`
- 多投毒率实验必须显式指定：`--poisoned-rate`
- 推荐显式指定 checkpoint，避免跨投毒率自动匹配串模型
